Writing a good question

You’re ready to ask a programming-related question and this form will help guide you through the process.

  • Summarize your problem in a one-line title.
  • Describe your problem in more detail.
  • Add "tags" which help relate your question to members of the community.
Title

Be specific and imagine you’re asking a question to another person.

Body

Introduce the problem and expand on what you put in the title. Minimum 30 characters.

Executable Code Snippet (Optional)

Add executable Java source code snippet. Readers can run this snippet directly from the question page.

Add up to 5 tags to describe what your question is about. Separated by commas.
